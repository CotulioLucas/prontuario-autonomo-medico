# Skill: Technical Translation

=================================================================

## Purpose

Traduzir documentação técnica para português preservando precisão, código, identificadores e links. Normalizar terminologia.

=================================================================

## When to Use

- Traduzir documentação técnica (prose) para português
- Revisar e padronizar termos técnicos em docs
- Manter código, identificadores e links inalterados na tradução

=================================================================

## Responsibilities

- Traduzir prosa
- Preservar código
- Preservar identificadores
- Preservar links
- Normalizar terminologia

=================================================================

## Prohibited

- Traduzir código ou identificadores
- Alterar links ou referências técnicas
